    ⚙️ Технологічний Стек

    Уся мережева взаємодія побудована на сучасних асинхронних патернах, що не блокують роботу сервера. На графіку представлено ключові бібліотеки Python, які забезпечують високу швидкодію та надійність проекту.

    • ⚡ asyncio & aiohttp: Асинхронні мережеві запити.
    • 🔌 websockets: Забезпечення зв'язку в реальному часі.
    • 📝 aiofile: Непомітне асинхронне логування.

    🛠 Інструкція з розгортання та використання

    Покроковий майстер налаштування. Перемикайте вкладки нижче, щоб пройти шлях від завантаження коду до запуску чату або CLI-утиліти. Структура створена для легкого копіювання команд.

    Крок 1: Завантаження коду

    Склонуйте репозиторій на свій комп'ютер:

    git clone https://github.com/Hrigoriu/webSocket_chat-exchange_pivatbank_bot.git
    cd webSocket_chat-exchange_pivatbank_bot

    Крок 2: Віртуальне середовище (Обов'язково)

    Для Windows:

    python -m venv venv venv\Scripts\activate

    Для Mac / Linux:

    python3 -m venv venv source venv/bin/activate

    Крок 3: Встановлення необхідних бібліотек

    Після активації віртуального середовища, необхідно встановити всі сторонні пакети, які використовуються у проекті. А саме:

    • aiohttp — для асинхронних мережевих запитів до API ПриватБанку.
    • websockets — для створення сервера та клієнта чату реального часу.
    • faker — для автоматичної генерації реалістичних українських імен користувачів.
    • aiofile та aiopath — для асинхронного запису історії команд (логування) у файл.

    Виконайте цю команду в терміналі, щоб встановити їх усі одночасно:

    python -m pip install aiohttp websockets faker aiofile aiopath
    💡

    Проект має інтерактивний чат реального часу. Кожен користувач отримує згенероване українське ім'я. У чаті працює бот.

    1. Запуск сервера

    python server.py

    2. Команди бота в чаті

    • Пишіть звичайний текст для спілкування.
    • exchange — показати курс валют за поточний день.
    • exchange 3 — показати курс за останні 3 дні.
    💻

    Консольна утиліта дозволяє швидко отримувати курси валют прямо в терміналі, без необхідності піднімати WebSocket сервер.

    Отримати базові курси (EUR, USD)

    python cli.py 2

    Отримати розширений список валют

    python cli.py 4 -c EUR USD PLN CHF
